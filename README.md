# Automation Environment Setup

This repository contains a `Makefile` to automate the installation of Appium and iOS testing tools.

## Prerequisites
- macOS (for iOS automation)
- Xcode installed

## Usage
1. Clone the repo: `git clone <https://github.com/BrandonLeeTech/automation-setup.git>`
2. Run setup: `make setup`
3. Verify environment: `make check`

## Locked Versions
- Appium: 3.1.2
- XCUITest Driver: 7.24.1
- UiAutomator2 Driver: 3.8.0
